# Thanks for choosing Halibut!

 Halibut is a light GTK web-browser. It is an ideal web browser if you are using GNOME or KDE on a low-power device.

## Dependinces:
- Linux - An ideal os.
- Python3.6+
- python3-gi
- urllib
- pyinstaller
- Virtual envrioments are not supported


Installation:
No compliation
